Quaternion

Defines a quaternion. Quaternions are used as rotations in both THREE.js and Ammo.js, and they avoid the problem of gimbal lock that euler angles have

Code example

let x = 0;
let y = 0;
let z = 0;
let w = 1;
const quat = new JS3DAmmo.Quaternion(x, y, z, w);

Constructor

Quaternion(x: Float, y: Float, z: Float, w: Float);


x - x value of the quaternion
y - y value of the quaternion
z - z value of the quaternion
w - w value of the quaternion

Properties

_x - x value of quaternion. Here for compatibility with three.js quaternion.
_y - y value of quaternion. Here for compatibility with three.js quaternion.
_z - z value of quaternion. Here for compatibility with three.js quaternion.
_w - w value of quaternion. Here for compatibility with three.js quaternion.

Methods

clone(): Quaternion

Clones this quaternion and returns the new one

conjugate(): this

calculates the conjugation for this quaternion

copy(q: Quaternion): this

copies quaternion q's values to this quaternion. workes with JS3DAmmo, THREE, and Ammo Quaternions

dot(q: Quaternion): Float

calculates the dot product of this quaternion and quaternion q

getAxisAngle(): Object: {axis: Vector3, angle: Float}

isAlmostEqualTo(q: Quaternion, delta: Float): Boolean

checks if this quaternion is equal to quaternion q within delta distance

isEqualTo(q: Quaternion): Boolean

checks for strict equality between this quaternion's values and quaternion q's values

gets the axis and angle around which this quaternion rotates

length(): Float

calculates the euclidean length of the quaternion

lengthSq(): Float

calculates the square of the euclidean length of the quaternion

normalize(): this

makes this quaternion be of unit length, but have same rotation as before

set(x: Float, y: Float, z: Float, w: Float): this

sets this quaternion's values to the inputted x, y, z, and w values

setFromArray(array: Array, offset: Integer): this

sets this vectors components from the array, starting at offset

setFromAxisAngle(axis: Vector3, angle: Float): this

sets this quaternion from the angle around the axis specified. assumes axis is normalized.

toString(): String

returns a string with this quaternion's values: '{x: (this x), y: (this y), z: (this z), w: (this w)}'
