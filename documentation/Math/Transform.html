Transform

Defines a Transform. Holder for a position and rotation

Code Example

const position = new JS3DAmmo.Vector3(0, 0, 0);
const rotation = new JS3DAmmo.Quaternion(0, 0, 0, 1);
const transform = new JS3DAmmo.Transform(position, rotation);

Properties

position - position of the transform
rotation - rotation of the transform

Methods

see Vector3 and Quaternion for derived properties

clone(): Transform

creates a new transform with the same position and rotation as this one

copy(t: Transform): this

copies transform t's position and rotation to this

normalize(rotation: Boolean, position: Boolean): this

normalizes rotation and/or position depending on what was inputted

set(px: Float, py: Float, pz: Float, qx: Float, qy: Float, qz: Float, qw: Float): this

sets position and rotation to inputted values
