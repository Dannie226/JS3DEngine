Vector2

Defines a 2D vector. Can be used as a Vector with a direction and magnitude, can be used as a point, or can be used as an arbitrary set of 2 numbers

Code Example

let x = 0;
let y = 0;
const vec = new JS3DAmmo.Vector2(x, y);

Constructor

Vector2(x: Float, y: Float);


x - x value of the vector
y - y value of the vector

Methods

add(a: Vector2): this

Adds vector a to this vector

addVectors(a: Vector2, b: Vector2): this

Sets this vector to be the sum of a and b

addScaledVector(v: Vector2, s: Float): this

Scales vector v by s and then adds v to this vector

clone():Vector2

returns a new Vector2 exactly the same as this vector

copy(v: Vector2): this

sets this vector to be the inputted one. Does work with JS3DAmmo.Vector2, THREE.Vector2, and Ammo.btVector2

directionTo(a: Vector2, b: Vector2): this

sets this vector to be the normalized direction from a to b

distanceTo(v: Vector2): Float

returns the euclidean distance from this vector to vector v

distanceToSquared(v: Vector2): Float

returns the euclidean distance squared (doesn't take square root at end). This is often easier to compute than the euclidean distance

dot(v: Vector2): Float

returns the dot product of this vector and vector v

getComponent(index: Integer): Float

gets a component of the vector:
x - index 0
y - index 1

invert(): this

Inverts this vector, ie: x = -x, y = -y

isAlmostEqualTo(v: Vector2, delta: Number): Boolean

returns if this vector is different from the other vector by at most delta in each direction

isEqualTo(v: Vector2): Boolean

checks for strict equality between this vector and v

length(): Float

computes the euclidean length of this vector

lengthSq():Float

computes the euclidean length squared of this vector. Often easier to compute than length()

manhattanDistanceTo(v: Vector2): Float

computes the manhattan distance from this vector to v

manhattanLength(): Float

computes the manhattan length of this vector

multiplyScalar(s: Float): this

multiplies each of this vectors components by s

normalize(): this

normalizes this vector, ie: makes the vector have the same direction, but length of 1

set(x: Float, y: Float): this

sets the components of this vector to the inputted x and y values

setFromArray(array: Array, offset Integer): this

sets this vectors components from the array, starting at offset

setLength(length: Float): this

sets this vector to be one that has length length and the same direction as the original one

setScalar(scalar: Float): this

sets each component of this vector to the scalar

sub(v: Vector2): this

subtracts vector v from this vector

subVectors(a: Vector2, b: Vector2): this

sets this vector to be a-b

toString(): String

returns a string: {x: (this vector x), y: (this vector y)}
