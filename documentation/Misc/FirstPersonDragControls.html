FirstPersonDragControls

creats a controls type meant for if PointerLockControls doesn't work

Code Example

const camera = world.cameras[0];
const renderer = world.renderer;
const controls = new JS3DAmmo.FirstPersonDragControls(camera,renderer.domElement);

Properties

object - object that has the rotation being applied to it
domElement - HTMLElement for the event handlers
maxPolarAngle - maximum polar angle of the object
minPolarAngle - minimum polar angle of the object
maxAzimuthAngle - maximum azimuthal angle of the object
minAzimuthAngle - minimum azimuthal angle of the object

methods

forward(distance: Float): void

moves the camera forward in the direction it is facing the specified amount

backward(distance: Float): void

moves the backward forward in the direction it is facing the specified amount

left(distance: Float): void

moves the camera left in the direction it is facing the specified amount

right(distance: Float): void

moves the backward right in the direction it is facing the specified amount
