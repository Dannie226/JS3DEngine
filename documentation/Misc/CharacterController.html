CharacterController

defines a way to move a character or body

Code Example

const character = new JS3DAmmo.Character({radius:1,height:2});
const camera = world.cameras[0];
const characterController = new JS3DAmmo.CharacterController({
  character:character,
  camera:camera
})

Properties

camera - camera in which the controller calculates direction for movement
cameraYOffset - if camera is in first person mode, the amount above the position of the character the camera is
character - RigidBody for the character
firstPerson - whether the character controller is first person or not
moveDir - object with left, right, forward, backward, and ableJump booleans
moveSpeed - speed of movement

Methods

forward(): void

changes move dir for forward with respect to the camera

backward(): void

changes move dir for backward with respect to the camera

left(): void

changes move dir for left with respect to the camera

right(): void

changes move dir for right with respect to the camera

updateMoveDirStart(keyCode: Integer, forCode: Integer, backCode: Integer, rightCode: Integer, leftCode: Integer): void

updates moveDir for starting

updateMoveDirStop(keyCode: Integer, forCode: Integer, backCode: Integer, rightCode: Integer, leftCode: Integer): void

updates moveDir for stopping

updatePosition(): void

updates velocity and position for character

jump(jumpVelocity: Float): Float

sets physics body y velocity to jump velocity
